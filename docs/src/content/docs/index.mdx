## 🎯 Tổng Quan Dự Án

**Calorie Web** là ứng dụng web hiện đại giúp người dùng:
- ✅ Ghi nhật ký calo từng ngày với 4 bữa ăn (Breakfast, Lunch, Dinner, Snack)
- ✅ Thêm/xóa nguyên liệu cho từng bữa, nhập số gram, tự động tính calories, protein, carbs, fat
- ✅ Tìm kiếm nhanh nguyên liệu với cơ sở dữ liệu
- ✅ Tính tổng calories từng bữa và cả ngày, hiển thị rõ ràng
- ✅ Lưu trữ tự động qua localStorage (Zustand), không mất dữ liệu khi reload
- ✅ Giao diện hiện đại với Tailwind CSS v4
- ✅ Biểu đồ thống kê với Recharts

**Đối tượng sử dụng:** Sinh viên, người tập gym, người quan tâm sức khỏe

## 📝 Type Definitions

**Files:** `types/index.ts`, `types/user.ts`

### Main Interfaces

```typescript
// Mục tiêu dinh dưỡng hàng ngày
interface MacroTarget {
  calories: number      // Daily calorie target (kcal)
  protein: number       // Daily protein target (g)
  carbs: number         // Daily carbs target (g)
  fat: number           // Daily fat target (g)
}

// Thông tin người dùng
interface UserProfile {
  name: string                        // Full name
  age: number                         // Age (years)
  weight: number                      // Weight (kg)
  height: number                      // Height (cm)
  goal: 'lose' | 'maintain' | 'gain'  // Health goal
  macroTarget: MacroTarget             // Daily targets
  currentStreak: number                // Current streak
  bestStreak: number                   // Best streak
}

// Một món ăn đã ghi
interface MealEntry {
  id: string
  name: string
  amount: number        // in grams
  calories: number
  protein: number
  carbs: number
  fat: number
}

// Nhật ký hàng ngày
interface DailyLog {
  date: string          // ISO date string
  breakfast: MealEntry[]
  lunch: MealEntry[]
  dinner: MealEntry[]
  snack: MealEntry[]
  water: number         // ml
  notes?: string
  totalCalories: number
  totalProtein: number
  totalCarbs: number
  totalFat: number
}
```

## 📦 State Management (Zustand)

### useProfileStore
**File:** `store/profileStore.ts` - Quản lý profile người dùng
- Auto-persisted to localStorage
- Sử dụng trong: `ProfileForm`, `DashboardPage`, stats

### useDiaryStore
**File:** `store/diaryStore.ts` - Quản lý nhật ký calo hàng ngày
- Auto-persisted to localStorage
- Sử dụng trong: `AddMealModalV2`, `DashboardPage`, diary page

### useAppStore
**File:** `store/useAppStore.ts` - Global app state
- Simple test store for global state

## 🛠️ Utility Functions

### lib/calc.ts - Calculations
- Activity levels & multipliers (1.2 - 1.9)
- Goal adjustments: lose (-500), maintain (0), gain (+300)
- Macro ratios based on goals
- BMR & TDEE calculations (Mifflin-St Jeor formula)

### lib/search.ts - Ingredient Search
- Server Action for ingredient searching
- Vietnamese text normalization
- Match scoring system
- Returns top N results

### lib/storage.ts - localStorage Utilities
- Safe serialization/deserialization
- Error handling
- Persist/restore objects

### lib/updateStreak.ts - Streak Calculation
- Check macro targets daily
- Update streak counter
- Track best streak

### lib/ingredients.json - Food Database
- Vietnamese ingredients with nutritional data
